# Arcgentic

Ask, Learn, Master.

Arcgentic is a multi-agent, agentic learning platform where users can:

- Ask questions about any topic.
- Upload reference documents to learn from.
- Generate learning content such as explanations, podcasts, and presentations.
- Interact with inline learning UIs generated by the learning agent.
- Track progress and earn skills when learning modules are completed.

## Monorepo Structure

- `apps/web`: Next.js frontend for the learning experience.
- `apps/agent_service`: Agent orchestration and content generation.
- `apps/user_service`: User, profile, and skill-progression backend services.
- `packages/*`: Shared workspace tooling and libraries.

## Development

Install dependencies:

```sh
pnpm install
```

Run all apps:

```sh
pnpm dev
```

Build all apps:

```sh
pnpm build
```
